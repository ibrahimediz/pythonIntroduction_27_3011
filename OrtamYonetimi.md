# * Ortam Oluşturma
    - Linux ve Mac <br>
        * python3 -m venv ["env"]
    - Windows<br>
        * python -m venv ["env"]
# * Ortamı Aktif Hale Getirmek
    - Linux ve Mac <br>
        * source env/bin/activate
    - Windows<br>
        * env\Scripts\activate
# * Ortamı Aktif Hale Getirmek
    - Windows<br>
        * env\Scripts\deactivate
# * Kütüphane Listesi Almak
    - pip freeze > requirements.txt
# * Kütüphane Listesi Yüklemek
    - pip install -r requirements.txt
